Your AI.

eVi is a local-first personal AI assistant. Chat with a model running on your machine, let it use tools, write code, generate images, run scheduled work, and drive your browser — from a terminal, a web app, or a native desktop window, all over one shared core. No cloud required.

pip install evi-assistant

One core. Every surface.

A single Python engine (evi/) drives a CLI, a web UI, and a desktop app — and talks to whatever local LLM you already run.

🖥️

Runs on your backend

LM Studio, Ollama, llama.cpp, or any OpenAI-compatible server. eVi auto-detects what's live and recommends a model that fits your hardware.

🧰

Real tools

Files, code execution, shell, web search, git, SQLite, PDF, OCR, vision, and voice — with a sandboxed run_python and granular per-tool permissions.

🧠

Memory & skills

Persistent cross-session memory, on-demand Skills (SKILL.md packets), and a hardware-aware model registry — all stored locally under ~/.evi.

🔌

MCP, both ways

Consume other MCP servers' tools, and run eVi as an MCP server exposing curated tools + memory resources to Claude Desktop, Cursor, and more.

🤝

Federation

Delegate heavy work to another eVi you own — keep a small model for chat and borrow your GPU box on the LAN, with no shared cloud.

⚡

Multi-agent

Ultracode fans work out across parallel subagents — decompose, solve, adversarially verify, and synthesize — for the tasks one context can't hold.

🛡️

Guardrails

Layered safety — regex → LLM judge → offline classifier — plus hooks, permission modes, trusted directories, and an MCP allowlist.

🔒

Yours, offline

Local-first by default. Your prompts, transcripts, and keys stay on your disk. Optional online model presets when you want them.

Three ways to run it

⌨️ Terminal

A fast Typer/Rich REPL. Chat, run tools, drive the whole agent from the command line.

evi chat

🌐 Web

A FastAPI + SSE app with streaming chat, settings, plugins, evals, and federation panels.

evi web

🪟 Desktop

A signed Tauri app for Windows, macOS, and Linux that wraps the web UI and self-updates.

Install

From PyPI

pip install evi-assistant
evi models recommend   ## what'll fit your box
evi chat

The distribution is evi-assistant; the import package and CLI stay evi. Requires Python 3.13+.

From source

git clone https://github.com/evi-assistant/evi-ai.git evi
cd evi
py -3.13 -m venv .venv
.venv\Scripts\activate        ## Windows
pip install -e ".[web,mcp,dev]"

Point it at a local backend (LM Studio / Ollama / llama.cpp) or run evi setup for a guided first run.

Download the desktop app

Signed installers for every platform, built from the same 1.0 release. The app self-updates from here on.

🪟 Windows.msi / .exe macOS.dmg 🐧 Linux.deb / .rpm / .AppImage

All builds on the latest release page. Installers are minisign-signed for the updater (OS code-signing is on the roadmap, so SmartScreen/Gatekeeper may warn on first run).

Extend it with Skills

A curated, open catalog of Agent Skills — author .docx/.xlsx/.pptx, triage a git repo, sweep for secrets. Install any of them in one line.

evi skill add docx-author
