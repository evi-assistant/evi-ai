# Anthropic Claude Features (Code + Chat/Desktop) — 2026

## Claude Desktop App (Three Surfaces)

- **Chat** — conversation surface for asking, drafting, thinking
- **Claude Code** — developer surface (write/run/ship code)
- **Cowork** — desktop assistant that organizes files, runs scheduled tasks

---

## Claude Code — Major Features

- **Redesigned desktop app** with parallel sessions and drag-and-drop workspace
- **Agent view** — manage multiple sessions from one CLI view; background agents, peek at status, jump back when input needed
- **Session sidebar** — filter by status/project/environment, group by project
- **Routines** — bundle prompt + repo + connectors into automations; run on schedule, API call, or GitHub events
- **Computer use** (Pro/Max) — Claude opens files, runs dev tools, clicks/navigates with no setup
- **Plan Mode** — structured roadmaps with varying planning depth
- **MCP server support** — extend with external tools/data
- **Hooks** — automate repetitive actions (formatting, batch changes)
- **CLAUDE.md** — markdown config preserving project context, rules, preferences
- **/goal** — set completion condition; Claude works toward it across turns
- **Dreaming** — scheduled review of past sessions, curates memory between runs
- **Auto mode** — auto-approves safe actions, blocks risky ones
- **SSH support** (macOS)
- **Subagents** — specialized agents (e.g., Explore, Plan) for delegated tasks
- **Scheduled tasks / cron** — recurring or one-shot prompts
- **Worktrees** — isolated git copies for parallel work
- **Remote triggers** — execute via API

## Claude Code — Minor Features

- Side chat shortcut (Cmd+;) for branching off a running task
- Integrated terminal for tests/builds
- In-app file editor for spot edits
- Rebuilt diff viewer for large changesets
- Expanded preview pane (HTML, PDFs, local app servers)
- Slash commands (`/help`, `/fast`, `/commit`, `/loop`, `/schedule`, etc.)
- Fast mode toggle (`/fast`) — same Opus 4.6 model, faster output
- Background tasks & TaskOutput
- TodoWrite for task tracking
- Skills system (user-invocable + auto-triggered)
- Status line customization
- Keybindings customization (`~/.claude/keybindings.json`)
- IDE extensions (VS Code, JetBrains)
- Web app at claude.ai/code

---

## Claude Chat (Desktop + Web) — Major Features

- **Projects** — persistent knowledge + custom instructions per workspace
- **Artifacts** — interactive code, charts, diagrams, mobile apps in side panel
- **Computer use** — screen interaction capability
- **Connectors / MCP** — integrate Gmail, Calendar, Linear, Jira, GitHub, etc.
- **Claude Skills** — domain-specific capabilities (docx, pptx, pdf, xlsx, etc.)
- **Memory** — cross-conversation persistent context
- **File uploads** — PDFs, images, docs, spreadsheets
- **Voice mode**
- **Research/Web search**
- **Code execution / analysis tool**
- **Cowork integration**

## Claude Chat — Minor Features

- Conversation history search
- Conversation forking/branching
- Custom styles (formal, concise, explanatory)
- Shareable conversation links
- Dark/light themes
- Keyboard shortcuts
- Model selector (Opus 4.6, Sonnet 4.6, Haiku 4.5)
- Extended thinking toggle
- Image generation
- Mobile apps (iOS/Android) syncing with desktop
